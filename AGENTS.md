# FAVOR Project Guidelines

## Skills
- `/design-system` - Typography sizes, colors, and component patterns

## Key Files
- `docs/DESIGN_SYSTEM.md` - Full design system documentation
- `src/app/globals.css` - Theme tokens and CSS component classes

## Design Rules
1. **Default text size**: `text-sm` (14px)
2. **Navbar**: `text-base` (16px)
3. **Labels**: `text-xs` uppercase tracking-widest
4. **Colors**: Use slate palette (900/700/500/400/300)
5. **Borders**: Always `border-slate-200`

## Tech Stack
- Next.js (App Router)
- Tailwind CSS v4
- TanStack Table
- Recharts
- Radix UI primitives
