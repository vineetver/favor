## What's New

- **Two chart types** - Network view and circular chord diagram
- **Smart layouts** - Charts organize themselves to show connections clearly
- **Filter by evidence** - Show only high-confidence connections
- **Handles big datasets** - Works smoothly with 1000+ protein connections

## How to Use

- **Click any protein** - See its connections in the table below
- **Filter connections** - Choose by experiment type or confidence level
- **Switch views** - Network chart or circular diagram
- **Find hub proteins** - Highly connected proteins are highlighted
