## What's New

- **All tissue data in one view** - No more jumping between 8 different tabs
- **Smart filtering** - Only shows high-confidence results (scores above 0.1)
- **Organized by importance** - Most significant findings appear first
- **Easy to scan** - Clear sections for each data type

## What You'll See

- **Tissue activity scores** - How active this variant is in liver, brain, blood
- **Regulatory elements** - Which DNA control regions this variant affects
- **Gene connections** - Which genes this variant might influence
- **Disease links** - Known connections to traits and diseases

## Before vs Now

**Before:** Check 8 separate tabs to understand tissue effects
**Now:** See everything in one organized summary

This saves time and makes it easier to understand how variants work across the body.
