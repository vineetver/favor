## Key Improvements

- **Intelligent Point Distribution**: Smart jittering algorithm prevents overlapping while maintaining categorical grouping
- **Click-to-Table Integration**: Click any data point to instantly view detailed information in the table below
- **Category-Based Filtering**: Interactive category buttons with color-coded visualization
- **Responsive Design**: Optimized for both desktop and mobile viewing

## Interactive Features

- **Hover Tooltips**: Rich tooltips showing trait, p-value, effect size, and gene information
- **Point Size Mapping**: Circle size reflects effect magnitude for quick visual assessment
- **Significance Indicators**: Special triangle markers for variants above statistical cutoffs
- **Cross-highlighting**: Synchronized highlighting between chart and table views
