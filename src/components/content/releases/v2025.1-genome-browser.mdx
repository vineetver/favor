## What's New

- **Mobile-friendly track picker** - Works great on phones and tablets
- **Help docs for every track** - Click any track to see what it shows and how to use it
- **Search tracks** - Find tracks by name or what they measure
- **Track collections** - Pre-made sets for common research tasks

  <source src="/videos/mobile-track.mov" type="video/mp4">

## Track Documentation

Each track now has detailed info including:
- What the data shows
- How it was collected
- How to interpret the results
- Links to original papers

## Coming Soon

- **Research track sets** - Pre-selected tracks for cancer research, rare variants, and other specialties
- **Community tracks** - User-submitted tracks with peer review
- **Personal libraries** - Save your favorite track combinations

Makes genome browsing much easier, especially for researchers new to the platform.
